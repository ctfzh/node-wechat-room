# node-wechat-room
A simple wechat room using node + express + socket.io ，
一个简单的基于nodejs express socket 的防微信聊天室：
> 功能包括

- 发送表情
- 发送压缩图片
- 系统管理员推送消息


##使用方法

- 下载代码，在文件目录  `npm install` 下载依赖
- 下载完成后，执行 `node server.js`, *注意端口冲突*
